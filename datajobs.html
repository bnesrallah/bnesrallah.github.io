Data Jobs Survey
Dashboard

Using Power Query Editor to transform data and PowerBI to provide data insights.

Introduction

This is a project aimed at gaining some basic insights into survey data on job satisifaction in various data job fields. I will focus on getting a better understanding of what the most important data skills are for data jobs, job and salary satisfaction, as well as what the average salary is for various job titles.

Setup

There was no special setup required for this project. I simply downloaded the project dataset and ensured that Microsoft Power BI was installed and Power Query Editor was setup correctly.

The Data

The raw data consists of responses obtained from about 700 respondants from a multitude of data-related jobs. There is a lot of great data to work with in this dataset and so much data cleaning could be done. But, in this project I will keep the data cleaning rather simple. I may use the data again in another project to delve into some more complicated data cleaning.

Although most of the survey data contains predetermined field values (the respondant has to select between several predetermined responses), there are some questions that allowed for "Other" to be selected and then freeform comments to be added. This makes for data that is not very clean and would take some significant work to do so. So, for this project I will roll up all the "Other" responses into one by using a delimiter in Power Query Editor.

Another interesting issue with the data arises due to the fact that the survey respondants were asked to provide their current salary range, rather than an approximate number. This makes that salary data difficult to use in its current format. So, to simply matters I have taken the average of the minimum and maximum values of the range and used the average as the respondant's approximate salary.

Survey Dashboard

Here is the finished dashboard containing a number of types of visualizations found in PowerBI. I have also applied some advanced formatting techniques to improve the appeal of the dashboard.
